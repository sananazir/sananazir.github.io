Other Useful Resources

  • Virginia Tech Shock & Compressible Flow Calculator Interactive calculator for normal shock, oblique shock, and isentropic flow relations.
  • AirfoilTools NACA 4-Digit Generator Interactive generator and database for airfoil coordinates.
  • UIUC Airfoil Coordinates Database Comprehensive database of airfoil coordinate sets maintained by University of Illinois.
  • XFLR5 Download Official site for downloading XFLR5, an airfoil and wing analysis tool.
  • NIST Atomic Spectra Database Reference spectral line data used in the GlowSpec plasma emission simulator.
  • Johns Hopkins Turbulence Database (JHTDB) Large-scale turbulence simulation datasets with web APIs for research and education.
  • SU2 CFD Open-source CFD solver from Stanford, widely used for aerodynamic shape optimization and flow simulation.
  • OpenFOAM Widely used open-source CFD solver with extensive libraries for multiphysics and industrial applications.
  • NASA Turbulence Modeling Resource (TMR) Reference database for turbulence models, validation cases, and CFD benchmarks.
  • NASA Technical Reports Server (NTRS) Archive of NASA's technical papers, reports, and research notes in aerospace and CFD.
